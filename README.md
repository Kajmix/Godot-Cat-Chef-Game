# Cat Cheff Restaurant
<img src="https://i.ibb.co/nsBTHqMD/Baner-1.png">
Flavortown project ^^
<h3>Description:</h3>
2D pixel art style game, where u serve customers by giving them their orders and buying upgrades in the restaurant.
<h3>Keymap:</h3>
<ul>
<li>W/up arrow - move up</li>
<li>S/down arrow - move down</li>
<li>A/left arrow - move left</li>
<li>D/right arrow - move right</li>
<li>E/Z - interact</li>
</ul>
<h3>Game Mechanics:</h3>
<ul>
<li>WSAD player movement with animations</li>
<li>Speed boost (milk)</li>
<li>Based player speed: 75</li>
<li>3 food types, 6 normal and 2 special clients</li>
<li>Random (sprites) clients, have random orders, with random (in range) rewards</li>
<li>Table upgrades, unlocking tables</li>
<li>Autosave, reset data, pause</li>
<li>Tutorial</li>
<li>UI: money, keys icon, speed boost icon, monologue</li>
<li>Music, SFX, particles, fade transitions</li>
</ul>
